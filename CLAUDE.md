# Stare project — Claude Code instructions

Never include a "Co-Authored-By: Claude" trailer or a "Generated with Claude Code" line (or any emoji signature) in commit messages. ## Code style

Never use em-dashes in user-visible copy. Use commas, parentheses, colons, or sentence breaks instead.

## Brand
